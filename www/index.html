Fermer cette fenêtre

Explications :


Cette page est servie par un petit serveur Web statique, entièrement écrit en CoffeeScript et fonctionnant sous NodeJS. Le but de cet exercice était d'apprendre et de comprendre le fonctionnement des requêtes HTTP et comment le serveur les traites.

Ce serveur est un projet réalisé durant le stage en entreprise validant mon Master en Conception & Développement informatique (au sein de l'entreprise Coqs en Pâte).

Les contraintes étaient de ne pas utiliser de librairies externes, ni d'utiliser la librairie "http" de NodeJS.

Cette première version du serveur a été écrite en 2 semaines, en partant des spécifications décrites dans la documentation W3C du protocole HTTP 1.0 et supporte les méthodes "HEAD" et "GET".

  • Composition technique de la page à afficher :


    La page est composée de plusieurs types de fichiers :

    • des fichiers HTML
    • des fichiers CSS
    • des fichiers JS
    • un fichier mp3
    • des fichiers vidéo intégrés dans une iframe
